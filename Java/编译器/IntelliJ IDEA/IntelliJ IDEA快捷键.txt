Ctrl + E				可以显示最近编辑的文件列表

Ctrl + F4				关闭当前文件

Ctrl + H / Alt + 8		显示类结构图
Ctrl + J				查看所有代码模板
Ctrl + N				查找类
Ctrl + Shift + N		查找文件

Ctrl + P				显示方法参数
Ctrl + Q				显示注释文档
Ctrl + R				替换文本
Ctrl + X				删除光标所在行
Ctrl + +/-				当前方法展开、折叠

Ctrl + Shift + Up/Down/Left/Right	调整窗口上下左右大小
